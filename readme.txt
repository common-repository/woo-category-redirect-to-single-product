== Description ==

Now you can take more control over your categories if you have some that only have one product.

With this plugin, your clients will be redirect to single product page when your client click on a category that only have one product.

More information and request: https://www.ohyeahdev.es/plugins/woo-category-redirect-single-product

== Installation ==

1. Visit your admin section and click Plugins -> Add New.  Then search for "WooCommerce Category Redirect to Single Product‚Äù.

2. After activate, the plugin is running.
